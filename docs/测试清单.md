## 五、快速冒烟测试脚本

在干净的 Windows 环境中（或有 Node.js 18+），可以执行以下快速验证：

```powershell
# 1. 语法检查
powershell -ExecutionPolicy Bypass -File .\scripts\check.ps1

# 2. 诊断（跳过 API 测试）
powershell -ExecutionPolicy Bypass -File .\doctor.ps1 -NoSaveReport -SkipApiTest

# 3. 安装 Claude Code（如有 Node.js）
powershell -ExecutionPolicy Bypass -File .\install.ps1 -Mode InstallOnly -SkipDisclaimer -NonInteractive

# 4. 再次诊断
powershell -ExecutionPolicy Bypass -File .\doctor.ps1 -NoSaveReport -SkipApiTest
```

在 WSL 环境中：

```bash
# 1. 语法检查
bash scripts/check.sh

# 2. 诊断（不安装任何东西）
bash install_wsl.sh --mode doctor --skip-api-test

# 3. 环境检测（仅检测）
# 脚本主菜单前会自动运行 check_environment
echo "1" | bash install_wsl.sh --yes --skip-api-test || true
```
