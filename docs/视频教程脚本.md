# 视频教程脚本（约 3 分钟）

> 录屏参考脚本。语速适中，面向小白用户。
> 建议使用 OBS 或系统自带录屏，分辨率 1920×1080。

---

## 开场 (0:00 - 0:10)

**画面**: 显示 ZIP 文件和解压后的文件夹

**旁白**:
"大家好，这是一个 Claude Code + DeepSeek 一键安装包的使用教程。
你只需要三步：解压、双击、粘贴 Key。"

---

## 第 1 步：解压 (0:10 - 0:20)

**画面**: 右键 ZIP → 全部解压缩 → 选择桌面 → 打开解压后的文件夹

**旁白**:
"下载 ZIP 后，右键选择「全部解压缩」，解压到桌面。
打开解压后的文件夹，你会看到「开始安装」「一键诊断」「恢复或卸载配置」三个入口。"

**重点镜头**: 鼠标指向「开始安装.cmd」

---

## 第 2 步：双击开始安装 (0:20 - 0:35)

**画面**: 双击「开始安装.cmd」→ 窗口弹出 → 显示免责声明

**旁白**:
"双击「开始安装.cmd」。如果弹出安全提示，点击「更多信息」→「仍要运行」。
阅读声明后，输入 Y 确认。"

**重点镜头**: 免责声明页面，强调「不提供 API Key」

---

## 第 3 步：获取 API Key (0:35 - 1:00)

**画面**: 浏览器自动打开 DeepSeek API Keys 页面 → 登录 → 创建 Key → 复制

**旁白**:
"脚本会自动打开 DeepSeek API Key 页面。
如果你还没有账号，用手机号注册一个，免费的。
登录后点击「创建 API Key」，输入名称后确认。
**重要：Key 只显示一次，立刻复制！**"

**重点镜头**: 高亮复制按钮，文字提示「只显示一次」

---

## 第 4 步：粘贴 API Key (1:00 - 1:15)

**画面**: 回到安装窗口 → 粘贴 Key（不显示）→ 回车

**旁白**:
"回到安装窗口，粘贴刚才复制的 Key。
**注意：粘贴时不会显示任何字符，这是安全保护。**
直接 Ctrl+V 粘贴，按回车即可。"

**重点镜头**: 光标处无字符显示，强调「不显示是正常的」

---

## 第 5 步：等待自动安装 (1:15 - 2:00)

**画面**: 自动滚动检测和安装进度

**旁白**:
"接下来脚本会自动完成：
环境检测 → Claude Code 安装 → DeepSeek 配置写入 → API 连接测试。
你只需要等待，不需要任何操作。"

**重点镜头**: 
- 环境检测通过（绿色文字）
- Claude Code 安装成功
- DeepSeek 配置写入成功
- API 测试通过

> 录制时可以加速播放（1.5x-2x），旁白正常语速。

---

## 第 6 步：安装完成 (2:00 - 2:20)

**画面**: 显示「安装流程全部完成」→ 安装报告路径

**旁白**:
"看到「安装流程全部完成」就说明一切就绪了。
安装报告保存在 reports 目录下。
桌面上还自动创建了一个 ClaudeCode-Test 测试目录。"

**重点镜头**: 绿色完成框 + 报告路径 + 桌面测试目录

---

## 第 7 步：运行 claude (2:20 - 2:40)

**画面**: 打开终端 → cd 到测试目录 → 运行 claude → 输入第一个问题

**旁白**:
"打开 PowerShell 或命令提示符，进入桌面上的 ClaudeCode-Test 目录。
运行 claude，然后输入你的第一个问题。
比如「请帮我写一个 hello world 网页」。"

**重点镜头**: claude 启动画面 + 对话示例

---

## 第 8 步：演示一键诊断 (2:40 - 2:55)

**画面**: 返回安装包目录 → 双击「一键诊断.cmd」→ 显示诊断报告

**旁白**:
"如果遇到任何问题，双击「一键诊断.cmd」。
它会自动检查系统环境并生成报告。
把报告文件发给技术支持即可，报告中不包含你的 API Key。"

**重点镜头**: 诊断运行过程 + report 文件位置

---

## 结尾 (2:55 - 3:00)

**画面**: 项目 LOGO/标题

**旁白**:
"安装包和教程链接在商品详情中。感谢观看！"

---

## 录制要点

| 项目 | 建议 |
|------|------|
| 分辨率 | 1920×1080 |
| 帧率 | 30fps |
| 编码 | H.264 |
| 音频 | 清晰无杂音 |
| 鼠标 | 显示点击效果（OBS 可设置） |
| 敏感信息 | **不要显示真实 API Key**，用测试 Key 或打码 |
| 速度 | 等待环节可加速 1.5x-2x |

## 分镜时间线

```
00:00  开场 + ZIP 展示
00:10  解压操作
00:20  双击开始安装
00:35  获取 API Key（浏览器操作）
01:00  粘贴 Key
01:15  等待安装（加速播放）
02:00  安装完成展示
02:20  运行 claude 演示
02:40  一键诊断演示
02:55  结尾
03:00  结束
```
