## 4. 检查不应包含的文件

```powershell
cd .\release-check\
```

以下命令必须无输出：

```powershell
# 不应包含内部文件
Get-ChildItem -Recurse -Include ".git", "logs", "backup", "reports", "release" -Directory
Get-ChildItem -Recurse "CLAUDE.md", ".gitignore", "leak-test.md", "report*.txt", "*.tmp", "*.log"
Get-ChildItem -Recurse -Filter "build-release.ps1"
Get-ChildItem -Recurse -Filter "simulate-user-release.ps1"

# 不应包含的文件名模式
ls -Recurse | Where-Object { $_.Name -match 'CLAUDE\.md|build-release\.ps1|simulate-user-release\.ps1|\.gitignore|leak-test|report.*\.txt' }
```

## 5. 检查 .ps1 文件 BOM

```powershell
$b = [System.IO.File]::ReadAllBytes(".\Start-Here.ps1")
$b[0..2] -join " "
```

结果必须是: `239 187 191`

对所有 .ps1 文件验证:

```powershell
Get-ChildItem -Recurse -Filter *.ps1 | ForEach-Object {
    $b = [System.IO.File]::ReadAllBytes($_.FullName)
    if ($b[0] -ne 239 -or $b[1] -ne 187 -or $b[2] -ne 191) {
        Write-Host "[FAIL BOM] $($_.Name)"
    } else {
        Write-Host "[OK] $($_.Name)"
    }
}
```

全部必须 [OK]。

---

## 6. 检查 .cmd 文件编码

```powershell
Get-ChildItem -Recurse -Filter *.cmd | ForEach-Object {
    $b = [System.IO.File]::ReadAllBytes($_.FullName)
    $hasBom = ($b[0] -eq 239 -and $b[1] -eq 187 -and $b[2] -eq 191)
    $nonAscii = ($b | Where-Object { $_ -gt 127 }).Count
    if ($hasBom -or $nonAscii -gt 0) {
        Write-Host "[FAIL] $($_.Name) BOM=$hasBom nonASCII=$nonAscii"
    } else {
        Write-Host "[OK] $($_.Name)"
    }
}
```

全部必须 [OK]。

## 7. PowerShell 5.1 AST parse

在 Windows PowerShell 5.1 中运行（不要在 PowerShell 7 中运行）：

```powershell
Get-ChildItem . -Recurse -Filter *.ps1 | ForEach-Object {
    $tokens = $null
    $errors = $null
    [System.Management.Automation.Language.Parser]::ParseFile($_.FullName, [ref]$tokens, [ref]$errors) | Out-Null
    if ($errors.Count -gt 0) {
        Write-Host "[FAIL] $($_.FullName)" -ForegroundColor Red
        $errors
    } else {
        Write-Host "[OK] $($_.FullName)" -ForegroundColor Green
    }
}
```

全部必须 [OK]。
