## 小白版：双击安装（推荐）

### 第 1 步：解压

下载 ZIP 文件后，右键 →「全部解压缩」→ 选一个目录（桌面也可以）。

解压后你会看到这些文件：

```
开始安装.cmd          ← 双击这个！
一键诊断.cmd          ← 有问题时双击
恢复或卸载配置.cmd    ← 换 Key 时双击
```

### 第 2 步：双击「开始安装.cmd」

找到 **`开始安装.cmd`**，双击运行。

> 如果 Windows 弹出 "Windows 保护了你的电脑" 提示，点击「更多信息」→「仍要运行」。

### 第 3 步：复制 API Key

脚本会自动打开 DeepSeek API Key 页面。

1. 注册/登录 DeepSeek（https://platform.deepseek.com）
2. 在左侧菜单找到「API Keys」
3. 点击「创建 API Key」
4. **复制生成的 Key**（只显示一次！）

### 第 4 步：回到窗口粘贴

回到安装窗口，粘贴 Key 后按回车。

> [WARN] **粘贴时不显示字符是正常的！** 这是安全保护，直接粘贴按回车即可。

### 第 5 步：等待检测和安装

脚本会自动：
- 检查你的系统环境
- 安装 Claude Code
- 写入 DeepSeek 配置
- 测试 API 是否可用
- 创建测试项目

你只需要等待，看到屏幕上的进度提示即可。

### 第 6 步：看到安装完成报告

安装完成后会显示：

```
安装流程全部完成！
```

同时在 `reports/` 目录下会生成一份安装完成报告。

### 第 7 步：运行 claude

打开 PowerShell 或命令提示符：

```powershell
cd "%USERPROFILE%\Desktop\ClaudeCode-Test"
claude
```

然后输入你的第一个问题，比如：

```
请读取 README.md，并帮我生成一个简单的 hello world 网页。
```

### 遇到问题？

双击 **`一键诊断.cmd`**，把项目目录下生成的 `report.txt` 发给技术支持。

## 准备工作

### 1. 确认系统

- Windows 10 或 Windows 11
- 查看方法：右键"此电脑" → "属性" → 查看 Windows 版本

### 2. 获取 DeepSeek API Key

1. 打开浏览器，访问 https://platform.deepseek.com
2. 点击"注册"，使用手机号注册账号
3. 登录后，左侧菜单找到 "API Keys"
4. 点击"创建新 Key"，输入名称（随便填，如 "claude-code"）
5. 复制生成的 Key（**只显示一次！请立即保存**）

> [WARN] Key 格式类似：`sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

### 3. 确保网络正常

- 确保能正常上网
- 如果在公司网络，确保没有阻止 `api.deepseek.com`

## 运行安装脚本（高级用法）

> 如果你习惯用命令行，可以不用 .cmd 文件，直接用 PowerShell。

### 方法 1：右键菜单

1. 打开项目文件夹
2. 按住键盘 `Shift` 键
3. 在文件夹空白处右键
4. 选择 **"在此处打开 PowerShell 窗口"** （Win11 可能是 "在终端中打开"）
5. 输入：

```powershell
powershell -ExecutionPolicy Bypass -File .\Start-Here.ps1
```

### 方法 2：手动打开 PowerShell

1. 按键盘 `Win + R`
2. 输入 `powershell` 回车
3. 输入 `cd` 然后空格，把项目文件夹拖进窗口
4. 按回车
5. 输入上面的运行命令

### 首次运行提示

> 如果看到红色提示 "无法加载文件..."，说明执行策略限制。
> 只需复制粘贴整条命令（包含 `-ExecutionPolicy Bypass` 部分）即可。

## 配置 DeepSeek API

安装脚本运行后，会显示免责声明。输入 `Y` 同意后：

1. 选择 `[1] 一键安装（推荐）`
2. 等待环境检测和 Claude Code 安装完成
3. 当提示输入 API Key 时：
   - 回到浏览器，复制你的 DeepSeek API Key
   - 回到窗口，按 `Ctrl+V` 粘贴
   - **注意：粘贴后不会显示任何字符，这是正常的安全保护**
   - 按回车确认
4. 脚本会自动测试 API 连接
5. 看到「安装流程全部完成」→ 搞定！

## 启动 Claude Code

### 在 PowerShell/命令提示符中使用

配置完成后，打开 PowerShell 或命令提示符：

```powershell
claude
```

首次启动可能需要几秒钟加载。

### 在测试项目中使用

安装脚本会自动在你的桌面创建 `ClaudeCode-Test` 测试目录：

```powershell
cd "%USERPROFILE%\Desktop\ClaudeCode-Test"
claude
```

### 在 VS Code 中使用

1. 打开 VS Code
2. 按 `Ctrl+Shift+P`
3. 搜索 "Claude Code"
4. 选择 "Claude Code: Open" 或直接输入问题

### 启动后

你会看到一个对话界面，可以直接用中文和 AI 对话，让它帮你写代码、改 bug、解释代码等。

## 日常使用

### 基本对话

```
> 帮我写一个 Python 脚本，读取 CSV 文件并输出统计信息
```

### 代码修改

```
> 重构这个函数，让它支持异步调用
```

### 项目级别的操作

```
> 在整个项目中找出所有使用了 console.log 的地方
```

### 注意事项

1. **模型说明**：你使用的是 DeepSeek 模型（通过 Claude 兼容接口），不是 Claude 官方模型
2. **功能差异**：部分 Claude 原生功能可能不可用（如图片分析），以 DeepSeek 实际支持为准
3. **费用**：每次对话会消耗 DeepSeek API 额度，请注意余额
4. **网络**：使用时需要保持网络连接

## 遇到问题

### 第 1 步：双击「一键诊断.cmd」

这会在项目目录生成一份 `report.txt` 最新诊断报告，并在 `reports/` 目录保留带时间戳的历史报告。

### 第 2 步：查看报告

用记事本打开报告，查看哪些项目是 ERROR 或 WARN。

### 第 3 步：联系技术支持

将报告文件发送给卖家，**不要发送你的 API Key**。

### 常见问题自助

| 问题 | 解决方法 |
|------|---------|
| 双击开始安装没有反应 | 先确认已完整解压 ZIP，再试 `Start-Install.cmd` 英文入口；仍不行请截图窗口内容发给卖家 |
| 输入 Key 不显示 | 这是安全保护，直接粘贴回车 |
| `claude` 命令不存在 | 关闭重开 PowerShell |
| 401 错误 | API Key 不对，重新获取 |
| 402 错误 | DeepSeek 余额不足，充值 |
| 429 错误 | 请求太频繁，等几分钟 |
| 5xx 错误 | DeepSeek 服务器问题，等官方修复 |
| 网络错误 | 检查网络/代理/VPN |
| 没有 VS Code 能用吗 | 可以，CLI 是核心交付 |

## 更新和卸载

### 更新 Claude Code

双击「开始安装.cmd」，选择 `[1] 一键安装`，脚本会检测已有版本并询问是否更新。

### 修改 API Key

双击「恢复或卸载配置.cmd」，或运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\configure-deepseek.ps1
```

### 恢复旧配置

双击「恢复或卸载配置.cmd」→ 选择从备份恢复。

### 完全卸载 Claude Code

```powershell
npm uninstall -g @anthropic-ai/claude-code
```

配置文件需要手动删除（或使用「恢复或卸载配置.cmd」）。
