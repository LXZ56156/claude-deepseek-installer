## 准备工作

### 1. 确认系统

- Windows 10 或 Windows 11
- 查看方法：右键"此电脑" → "属性" → 查看 Windows 版本

### 2. 获取 DeepSeek API Key

1. 打开浏览器，访问 https://platform.deepseek.com
2. 点击"注册"，使用手机号注册账号
3. 登录后，左侧菜单找到 "API Keys"
4. 点击"创建新 Key"，输入名称（随便填，如 "claude-code"）
5. 复制生成的 Key（**只显示一次！请立即保存**）

> ⚠️ Key 格式类似：`sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

### 3. 确保网络正常

- 确保能正常上网
- 如果在公司网络，确保没有阻止 `api.deepseek.com`

## 运行安装脚本

### 方法 1：右键菜单（推荐）

1. 打开项目文件夹
2. 按住键盘 `Shift` 键
3. 在文件夹空白处右键
4. 选择 **"在此处打开 PowerShell 窗口"** （Win11 可能是 "在终端中打开"）
5. 在弹出的蓝色窗口中输入：

```powershell
powershell -ExecutionPolicy Bypass -File .\install.ps1
```

6. 按回车执行

### 方法 2：手动打开 PowerShell

1. 按键盘 `Win + R`
2. 输入 `powershell` 回车
3. 输入 `cd` 然后空格，把项目文件夹拖进窗口
4. 按回车
5. 输入上面的运行命令

### 首次运行提示

> 如果看到红色提示 "无法加载文件..."，说明执行策略限制。
> 只需复制粘贴整条命令（包含 `-ExecutionPolicy Bypass` 部分）即可。

## 配置 DeepSeek API

安装脚本运行后，会显示免责声明。输入 `Y` 同意后：

1. 选择 `[2] 安装 Claude Code 并配置 DeepSeek API`
2. 等待 Claude Code 安装完成
3. 当提示输入 API Key 时：
   - 回到浏览器，复制你的 DeepSeek API Key
   - 回到 PowerShell，按 `Ctrl+V` 粘贴
   - **注意：粘贴后不会显示任何字符，这是正常的安全保护**
   - 按回车确认
4. 脚本会自动测试 API 连接
5. 看到 `✅ 安装流程完成！` → 搞定！

## 启动 Claude Code

### 在 PowerShell 中使用

配置完成后，**关闭当前 PowerShell 窗口**，重新打开一个新的。

在新的 PowerShell 窗口中：

```powershell
claude
```

首次启动可能需要几秒钟加载。

### 在 VS Code 中使用

1. 打开 VS Code
2. 按 `Ctrl+Shift+P`
3. 搜索 "Claude Code"
4. 选择 "Claude Code: Open" 或直接输入问题

### 启动后

你会看到一个对话界面，可以直接用中文和 AI 对话，让它帮你写代码、改 bug、解释代码等。

## 日常使用

### 基本对话

```
> 帮我写一个 Python 脚本，读取 CSV 文件并输出统计信息
```

### 代码修改

```
> 重构这个函数，让它支持异步调用
```

### 项目级别的操作

```
> 在整个项目中找出所有使用了 console.log 的地方
```

### 注意事项

1. **模型说明**：你使用的是 DeepSeek 模型（通过 Claude 兼容接口），不是 Claude 官方模型
2. **功能差异**：部分 Claude 原生功能可能不可用（如图片分析），以 DeepSeek 实际支持为准
3. **费用**：每次对话会消耗 DeepSeek API 额度，请注意余额
4. **网络**：使用时需要保持网络连接

## 遇到问题

### 第 1 步：运行诊断脚本

```powershell
powershell -ExecutionPolicy Bypass -File .\doctor.ps1
```

这会生成一份 `report.txt` 文件。

### 第 2 步：查看报告

用记事本打开 `report.txt`，查看哪些项目是 ❌ 或 ⚠️。

### 第 3 步：联系技术支持

将 `report.txt` 发送给卖家，**不要发送你的 API Key**。

### 常见问题自助

| 问题 | 解决方法 |
|------|---------|
| `claude` 命令不存在 | 关闭重开 PowerShell |
| 401 错误 | API Key 不对，重新获取 |
| 402 错误 | DeepSeek 余额不足，充值 |
| 429 错误 | 请求太频繁，等几分钟 |
| 5xx 错误 | DeepSeek 服务器问题，等官方修复 |
| 网络错误 | 检查网络/代理/VPN |

## 更新和卸载

### 更新 Claude Code

```powershell
npm install -g @anthropic-ai/claude-code@latest
```

### 修改 API Key

```powershell
powershell -ExecutionPolicy Bypass -File .\configure-deepseek.ps1
```

### 恢复旧配置

```powershell
powershell -ExecutionPolicy Bypass -File .\uninstall-config.ps1
```

选择 `[1] 从备份恢复配置`。

### 完全卸载 Claude Code

```powershell
npm uninstall -g @anthropic-ai/claude-code
```

配置文件需要手动删除（或使用 `uninstall-config.ps1`）。
